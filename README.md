# 🚗 Circular Car Parking for Metro Mall

## 📌 Project Overview
This project is an automated multi-level circular car parking system designed to reduce space usage and improve parking efficiency in metro cities.

## ⚙️ Features
- Automated car parking system
- Space optimization
- Microcontroller-based control
- LCD display for status
- Sensor-based detection

## 🛠️ Components Used
- Microcontroller (8051)
- DC Gear Motor
- IR Sensor
- Relay Module
- LCD Display
- Power Supply Unit

## 🧠 Working Principle
The system uses sensors and a microcontroller to detect vehicle presence and automatically rotate the circular platform to allocate parking space efficiently.
